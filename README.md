# Project Name: DigiTools - Premium Digital Tools Marketplace

##  Description:
DigiTools is a modern web application where users can explore and purchase premium digital tools. It provides a smooth user experience with interactive UI, cart functionality, and real-time feedback using notifications.


##  Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- React Toastify



##  Features
- Add to Cart functionality with real-time count update
- Toggle between Products and Cart with dynamic UI
- Instant feedback using toast notifications
- Smooth tab switching between Products and Cart
